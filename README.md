# Pixel Manipulation for Image Encryption - PRODIGY_CS_02

## Description
A Python-based image encryption and decryption tool using pixel manipulation and XOR operations.

## Features
- Encrypt images
- Decrypt images
- Uses XOR encryption
- Simple and secure image processing

## Technologies Used
- Python
- Pillow Library

## How It Works
The program modifies image pixel values using XOR operations with a secret key. Applying the same key again decrypts the image back to its original form.

## How to Run

```bash
python image_encryptor.py
```
